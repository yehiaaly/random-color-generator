# Random Color Generator

A React application that generates random colors with their hex and rgb codes. Perfect for designers, developers, and anyone who needs color inspiration.

## Overview

The Random Color Generator is a simple yet powerful tool that:

- Generates random colors with a single click
- Displays the hex and rgb codes for a generated color
